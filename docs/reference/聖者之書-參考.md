# 創世紀聖者之書 — 萃取參考(p.15–p.28)

## 每頁主題

#### p.15 — 蓋瑞特設計創世紀四代:不創宗教爭議,以「智慧寶典(終極智慧)」連串任務為主軸;童話《綠野仙蹤》啟發八德來源。
#### p.16 — 八德源自三原動力(真理/愛/勇氣)的組合;六角星硬幣圖案、Ankh 護身符的由來與設計屬性測驗方式。
#### p.17 — 屬性決定與職業對應規則;對話方式(Name/Job/口令);神秘數字(八德/八城/八地牢)與飛行工具設定。
#### p.18 — 更真實的魔法系統(藥材成分概念);四代破紀錄與商業成績、獲獎紀錄。
#### p.19 — 第六章「招兵買馬」:第一代程式團隊招募(Origin 成立、各程式設計師加入)。
#### p.20 — 南遷德州;Lord British 失蹤的劇情設計問題與 Blackthorn(布雷克松)篡位設定。
#### p.21 — Blackthorn 劇情的玩家反應;敏感話題(離職員工事件、抓小孩進退兩難測試);魔法咒語(an / nox)。
#### p.22 — 居民作息時間真實化;地面景觀 512 種式樣;地底世界(Underworld)。
#### p.23 — 程式設計師的彩蛋(特殊按鍵);測試遊戲流程;四代年度獲獎。
#### p.24 — 第七章「重大的改革」:國際化(日本市場、Sierra 授權爭議);五代媒體曝光;構想「錯誤的預言」與天人合一。
#### p.25 — 知識寶典下沉造成翼魔(Gargoyle)世界劇變;六代劇情(黑暗時期、預言書 Book of Prophecies);圖塊增至 2048 種。
#### p.26 — 六代物理屬性真實化(重量/大小);指令簡化為「使用 Use」;地圖式樣 256×256;World Editor / Map Editor 工具。
#### p.27 — World Editor 用於蠻荒帝國、火星之旅;七代開發 Terramorph Draw(地形繪製機)。
#### p.28 — 第八章「創世紀第六代的製作過程」:Origin 座右銘;六代程式設計小組成員介紹。

## 數值 / 清單 / 對照表

### 八德(Eight Virtues)— 由三原動力組合
- 三原動力:真理(Truth)、愛(Love)、勇氣(Courage)
- 誠實(Honesty)= 真理
- 同情(Compassion)= 愛
- 勇敢(Valor)= 勇氣
- 正義(Justice)= 真理 + 愛
- 犧牲(Sacrifice)= 愛 + 勇氣
- 榮譽(Honor)= 真理 + 勇氣
- 靈性(Spirituality)= 真理 + 愛 + 勇氣(三者皆具)
- 謙虛(Humility)= 無任何原動力(六角星中央小圓)
- 驕傲(Pride)= 惡德(代表驕傲的城市早已被毀)

### 神秘數字「八」(p.17)
- 八項美德
- 八座城市
- 八個地牢
- 八句讚詞(Mantra)
- 八個標記(Rune)
- 小說章節數每章 8(片頭/簡介呼應)
- (典故:影集《Eight is Enough》)

### 屬性 ↔ 職業對應(p.17)
| 偏重屬性 | 對應職業 |
|---|---|
| 誠實(真理) | 智學家 / 法師(Mage,Intelligence 智慧高) |
| 勇敢(勇氣) | 武士(Fighter,Strength 力量高) |
| 同情(愛) | 牧師(Cleric / Bard?,Wisdom?) |
| 敏捷(Dexterity 高) | 吟遊詩人(Bard) |

> 註:屬性名詞 — 智慧(Intelligence)、力量(Strength)、智慧/善良(Wisdom)、敏捷(Dexterity);職業 — 法師(Mage)、武士(Fighter)、吟遊詩人(Bard)。

### 角色屬性點數
- 一系列關於八項美德的遭遇問答,共 63 種不同組合
- 累積總分:每答一次美德加分,計算 63 種問答後得分,推算各屬性

### 飛行/地圖/規模數值
- 第一代飛行船(帆船型):依風力航行,受速度/風向影響。
- Sosaria(四代大陸):第三代世界的 16 倍大。
- 地面式樣(tile 種類):四代 256 種 → 五代 512 種 → 六代 2048 種圖塊式樣。
- 六代地圖式樣:256×256 圖塊組成的地圖(地面);地底 1024×1024。
- (p.21) 地圖標記:大深淵標為 OJ, OJ(Orange Juice 惡搞)。

### 魔法咒語語法(p.21)
- 由古代讚活咒語(藥材成分概念)演化的魔法音節組合。
- 範例:`an` = 反義(否定);`nox` = 表示毒 → `an nox` = 解毒魔法的咒語。
- 咒語 = 將意思的音節依規則組合。
